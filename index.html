Understanding Functions in Python

A function is a reusable block of code that performs a specific task. Functions help us follow the DRY principle — Don't Repeat Yourself — making programs modular and easier to maintain.

def greet(name):
    """Greet a person by name in Swahili."""
    message = f"Habari, {name}! Karibu sana."
    return message

print(greet("Amara"))
# Output: Habari, Amara! Karibu sana.

Function Parameters & Arguments

Functions accept parameters allowing them to work with different data. Python supports positional, keyword, default, and variable-length arguments.

def calculate_grade(score, total=100, student="Student"):
    percentage = (score / total) * 100
    grade = "Distinction" if percentage >= 70 else "Pass"
    return f"{student}: {percentage:.1f}% — {grade}"

print(calculate_grade(85, student="Amara"))
# Amara: 85.0% — Distinction

Scope: Local vs Global Variables

Scope refers to where a variable can be accessed. Variables inside a function are local; variables outside are global.

💡 African Tech Insight: Scope in programming mirrors many African governance structures — decisions at the local (village) level don't automatically affect the global (national) level. Understanding boundaries prevents unintended side-effects in both governance and code.
